# python-bootcamp-projects
Here are my projects that I am doing in my 100-days of Code Pro Python Bootcamp. 

## Projects Included

### 🧮 CALCULATOR
A simple command-line claculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

### ♦️ BLACKJACK GAME
A python-based Blackjack game where:
- User playes against the computer
- Cards are randomly generated
- Basic blackjack rules are implemented

### 🔢 GUESS THE NUMBER GAME
A fun game where:
- The computer generates a random number
- The user tries to guess it
- Hints like "Too High" or "Too Low" are given

### 🪢 Hangman Game
A classic word guessing game where:
- The player guesses letters of a hidden word
- Limited attempts are given
- Game ends when the word is guessed or attempts run out

### 📊 Higher Lower Game
A comparison-based game where:
- Player guesses which option has a higher value
- Score increases with correct guesses
- Game continues until a wrong guess

## 🚀 How to Run

1. Make sure Python is installed
2. Clone this repository:
   git clone https://github.com/devakisan-hub/python-bootcamp-projects
3. Navigate to the folder and run:
   python filename.py
